# Islamic Pedia

Penulis: Fauzan January

Islamic Pedia adalah add-on NVDA yang dikembangkan untuk membantu tunanetra Muslim dalam mengakses informasi Islami serta menunjang kebutuhan ibadah sehari-hari. Add-on ini menyediakan jadwal sholat yang akurat berdasarkan lokasi pengguna, arah kiblat, kalender Hijriyah, ensiklopedia Islam, serta berbagai fitur lainnya yang akan terus dikembangkan secara berkelanjutan.
Saat ini Islamic Pedia hanya mendukung wilayah Indonesia. Dukungan untuk negara lain akan dipertimbangkan pada pengembangan berikutnya.

## Fitur

- Jadwal Sholat lima Waktu beserta Waktu Tambahan (Imsak, Terbit, dan Dhuha).
- Notifikasi Waktu Sholat dan Waktu Tambahan, Dilengkapi Pra-pengingat serta Audio yang Dapat Disesuaikan.
- Informasi Arah Kiblat Berdasarkan Lokasi Pengguna (Ditampilkan dalam Derajat dan Arah Mata Angin).
- Pencarian Masjid Terdekat (Radius 3km) dengan opsi navigasi Google Maps.
- Kalender Hijriyah dengan Sistem Pergantian Hari pada Waktu Maghrib sesuai Ketentuan Syariat Islam.
- Ensiklopedia Islami (Wikipedia).
- **Baru:** Penyesuaian Tanggal Hijriyah Manual (+/- 2 hari) untuk sinkronisasi rukyat.
- **Baru:** Pilihan Metode Perhitungan Sholat (Kemenag RI, MWL, ISNA, dll).
- **Baru:** Pilihan Mazhab Ashar (Syafi'i/Hanafi).

## Panduan Penggunaan

Islamic Pedia menggunakan sistem **Mode Perintah (Command Layer)** untuk menghemat shortcut keyboard agar tidak bentrok dengan add-on lain. Berikut ini panduan penggunaannya:

1. Tekan `NVDA+Shift+I` untuk masuk ke mode Islamic Pedia.
2. Tekan B atau F1 untuk melihat bantuan dan daftar perintah pengunaan.
3. Tekan P untuk masuk ke menu pengaturan (Atur lokasi dan notifikasi pengingat).
4. Shortcut untuk fitur lainnya baca di bantuan.
5. Tekan `Esc` untuk keluar dari mode.

## Shortcut (Pintasan)

Shortcut Utama Untuk Masuk ke Mode Perintah (command layer): `NVDA+Shift+I`.

- **B** atau **F1** - Menampilkan bantuan dan daftar perintah pengunaan.
- **P** - Membuka menu Pengaturan.
- **J** - Menampilkan jadwal sholat hari ini.
- **K** - Menampilkan arah kiblat.
- **M** - Mencari masjid terdekat (radius 3km).
- **T** - Menampilkan tanggal Hijriyah hari ini.
- **I** - Mencari topik di Ensiklopedia Islami.
- **Spasi** - Menghentikan audio (Adzan/Pengingat) yang sedang berbunyi.
- **Esc** - Keluar dari mode Islamic Pedia.

## Pengaturan

Anda dapat menyesuaikan berbagai opsi melalui menu Pengaturan (Tekan `P` di dalam mode perintah):

- Cari dan pilih kota tempat tinggal Anda untuk akurasi jadwal pada tab **Lokasi**.
- Atur mode notifikasi (Suara, Bicara, Keduanya, atau Mati) untuk setiap waktu sholat pada tab **Notifikasi**.
- Pilih file audio pengingat yang sesuai keinginan Anda pada tab **Notifikasi**, audio boleh berbeda untuk setiap waktu sholat.
- **Pengingat Awal**: Atur durasi pengingat awal (misal: 10 menit sebelum waktu masuk) agar Anda bisa bersiap-siap.
- Sesuaikan Metode Perhitungan, Mazhab Ashar, dan Koreksi Tanggal Hijriyah pada tab **Lanjutan**.
